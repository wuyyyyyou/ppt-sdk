# Triage Labels

工程技能使用五个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的状态字符串。

| Label in mattpocock/skills | Label in our tracker | Meaning |
| -------------------------- | -------------------- | ------- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格已明确，可交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当某个技能提到 triage role，例如“apply the AFK-ready triage label”，使用本表右侧对应的状态字符串。

如需改成本仓库自己的词汇，编辑右侧 `Label in our tracker` 一列。
