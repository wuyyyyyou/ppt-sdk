# 双方对比主题 token

这个模板使用语义化对比 token，不使用颜色名作为主题契约。`sideA` 和`sideB` 表示两个被比较的位置，实际颜色可以是任意品牌色或主题色，不固定为红色或蓝色。

## 颜色 token

- `canvas`：整页画布背景。
- `surface`：浮层、半透明面板等表面背景。
- `card`：重复卡片、表格单元格背景。
- `textPrimary`：主标题和正文文字。
- `textMuted`：次级标签、说明文字。
- `textSubtle`：页脚、来源说明、坐标轴和低强调元信息。
- `textInverse`：强色块上的反白文字。
- `stroke`：常规边框和分割线。
- `strokeSoft`：弱分割线、非激活轨道。
- `grid`：图表网格和背景网格。
- `tableGrid`：对比表格网格线。
- `comparison`：对比框架、汇总结论、中心关系的主色。
- `comparisonSoft`：对比语义的柔和装饰色。
- `comparisonTint`：对比语义的低对比背景。
- `comparisonBorder`：对比语义边框。
- `sideA`：第一个对比对象的主色。
- `sideATint`：第一个对比对象的浅色背景。
- `sideABorder`：第一个对比对象的边框色。
- `sideB`：第二个对比对象的主色。
- `sideBTint`：第二个对比对象的浅色背景。
- `sideBBorder`：第二个对比对象的边框色。
- `neutral`：不归属于任一对比对象的中性强调色。
- `neutralTint`：中性浅色背景，也用于预测区间等弱强调区域。
- `neutralBorder`：中性弱边框。
- `statusWarning`：风险、警告、负向状态。
- `statusSuccess`：成功、正向状态。
- `chart1` 到 `chart6`：通用图表 fallback 色板。
- `chartSectorAgriculture`、`chartSectorIndustry`、`chartSectorServices`：产业结构图表颜色。
- `chartAgeYoung`、`chartAgeWorking`、`chartAgeElderly`：年龄结构图表颜色。

## 阴影 token

- `card`：重复卡片阴影。
- `panel`：大面板阴影。
- `soft`：浮动图例、柔和表面阴影。
