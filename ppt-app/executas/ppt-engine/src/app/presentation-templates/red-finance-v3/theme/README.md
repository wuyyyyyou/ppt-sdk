# red-finance-v3 主题 token

## 颜色 token

- `canvas`：整页幻灯片背景。
- `surface`：主要内容承载面。
- `panel`：强调面板或较高层级容器背景。
- `card`：重复卡片背景。
- `textPrimary`：标题和正文主文字。
- `textSecondary`：普通内容面上的次级文字。
- `textMuted`：标签、说明、弱化文本。
- `textSubtle`：坐标轴、页脚、低强调元信息。
- `textInverse`：需要在浅色或强调色背景上使用的反色文字。
- `accent`：主强调色。
- `accentText`：`accent` 背景上的文字颜色。
- `accentStrong`：更强的强调色，用于柱、线、激活状态等。
- `accentSoft`：柔和强调填充，用于高亮卡片或图标底。
- `accentTint`：低对比度强调背景。
- `accentBorder`：强调色边框或分隔线。
- `accentMutedText`：强调浅底上的可读文字。
- `stroke`：标准边框和图表网格线。
- `strokeSoft`：更弱的分隔线和未激活轨道。
- `axis`：图表坐标轴。
- `iconMuted`：非激活或次级图标描边。
- `rowAccent`：表格或矩阵行高亮背景。
- `chart1` 到 `chart6`：默认图表序列颜色。

## 阴影 token

- `card`：普通卡片阴影。
- `panel`：大型面板阴影。
- `accent`：强调色块或高亮元素阴影。
