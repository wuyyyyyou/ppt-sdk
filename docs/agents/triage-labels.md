# Triage Labels（triage 标签映射）

这些技能使用五个标准 triage 角色。这个文件把它们映射到本仓库 issue tracker 里实际使用的标签名。

| mattpocock/skills 里的角色 | 本仓库里的标签名 | 含义 |
| -------------------------- | ---------------- | ---- |
| `needs-triage`             | `needs-triage`   | 维护者需要先看一眼 |
| `needs-info`               | `needs-info`     | 还在等提问者补充信息 |
| `ready-for-agent`          | `ready-for-agent` | 信息完整，可以交给 AFK agent |
| `ready-for-human`          | `ready-for-human` | 需要人工实现 |
| `wontfix`                  | `wontfix`        | 不会处理 |

当技能提到某个角色时（比如“apply the AFK-ready triage label”），就用这张表里对应的标签名。

如果你们实际用的词不一样，改右边那一列就行。
